# HUMAN.zip

인터넷 속 당신은 어떤 인간인가요?

HUMAN.zip은 사용자의 인터넷 및 SNS 사용 습관을 기반으로 인터넷 인간 유형을 분석해주는 감성형 테스트 웹사이트입니다.

사용자는 여러 질문에 답하며 자신의 인터넷 성향을 확인할 수 있고, 유형별 특징, 추천 음악, 대표 색감 등의 결과를 카드 형식으로 확인할 수 있습니다.

주요 기능
인터넷 인간 유형 테스트
유형별 결과 카드 출력
전체 유형 모아보기
다크모드 / 라이트모드
결과 저장 기능 (localStorage)
반응형 웹 지원

유형
🌙 새벽감성 과몰입형
📱 릴스 무한 스크롤형
☕ 카페인 연명형
📅 계획만 완벽형
🎧 현실도피 플레이리스트형
🫠 인터넷 고인물형
